 ## Prerequisites
- Python 3.6
- PyTorch 0.4.0
- Numpy (>=1.12)
- Maybe more, just use `pip install` if you get an error.
 ## Dataset
All the data are with code. They are in `data` folder.
 ## To run
Issue the command for ingredient embedding:
```
python run.py
```
 To set dataset specific hyperparameters modify `Config.py`.
